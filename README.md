# EduManage - Cổng Thông Tin Sinh Viên

## Danh Mục
Frontend, Ứng Dụng Web

## Giới Thiệu
EduManage là giao diện cổng thông tin sinh viên, cung cấp các chức năng quản lý thông tin học tập, điểm số, lịch thi, học phí và hoạt động rèn luyện.

## Chức Năng
- Dashboard tổng quan
- Xem thông tin cá nhân
- Xem kết quả học tập (điểm số)
- Lịch thi
- Thời khóa biểu
- Quản lý học phí
- Đăng ký học phần và rút học phần
- Điểm rèn luyện và hoạt động rèn luyện
- Kiểm tra tốt nghiệp
- Thông báo

## Công Nghệ Sử Dụng
- **Frontend**: HTML5, CSS3, JavaScript
- **Dữ liệu**: JSON (data.json)

## Yêu Cầu Hệ Thống
- Trình duyệt web (Chrome, Firefox, Edge)

## Cài Đặt
```bash
git clone <repository-url>
cd EduManage
```

## Chạy Ứng Dụng
Mở file `dashboard.html` trên trình duyệt hoặc sử dụng Live Server.
